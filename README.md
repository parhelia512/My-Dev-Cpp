# My Own Dev-C++

*A new and improved fork of Orwell Dev-C++*

## Main Features

- MinGW-W64 GCC 8.1.0 32/64bit
- Syntax highlighting
- Code completion
- Code insight
- Editable shortcuts
- GPROF profiling
- GDB debugging
- AStyle code formatting
- Devpak IDE extensions
- External tools
